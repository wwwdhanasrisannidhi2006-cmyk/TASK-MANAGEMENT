Task Management Application

A full-stack web application designed to create, update, and track daily tasks efficiently.

🚀 Features

*   **User Authentication & Authorization**: Secure registration, login, and protected routes.
*   **CRUD Operations**: Complete functionality to Create, Read, Update, and Delete tasks.
*   **Real-time Updates**: Real-time task modifications synchronized instantly (Optional: Powered by WebSockets).
*   **Responsive Design**: Optimized for a seamless user experience across mobile, tablet, and desktop screens.

---

🛠️ Tech Stack

**Frontend:**
* HTML5 / CSS3 / Javascript
